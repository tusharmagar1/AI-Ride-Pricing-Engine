# 🚕 AI Dynamic Ride Pricing & Revenue Optimization System

A professional Streamlit-based web application that simulates an
AI-powered ride pricing engine.\
This dashboard dynamically adjusts ride prices based on real-time
demand, driver availability, and customer ratings to maximize revenue.

------------------------------------------------------------------------

## 📌 Project Overview

This project demonstrates how dynamic pricing works in ride-booking
platforms.\
It uses demand-supply logic and performance metrics to optimize ride
prices and estimate total revenue.

The system also includes a secure login feature and interactive
analytics visualization.

------------------------------------------------------------------------

-   🔐 Secure Login System (Session-Based Authentication)

------------------------------------------------------------------------

## 🧠 Pricing Optimization Logic

The ride price is adjusted based on:

-   Rider-to-driver ratio (Demand vs Supply)
-   Surge pricing when demand is high
-   Price reduction when driver supply exceeds demand
-   Additional increase for high-rated rides

This simulates real-world pricing models used in ride-hailing platforms.
